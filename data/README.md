# data/

This directory is **gitignored**. It is the shared bind-mount volume for all Docker containers.

## Expected structure:

```
data/
├── docs/                     # Source PDF files
│   ├── rtgs.pdf
│   ├── legal_circular_1.pdf
│   └── legal_circular_2.pdf
│
├── docs_images/              # PDF → page PNGs (one subdir per doc)
│   ├── rtgs/
│   │   ├── page_001.png
│   │   ├── page_002.png
│   │   └── ...
│   ├── legal_circular_1/
│   └── legal_circular_2/
│
├── pipeline_output/          # Generated by extraction_service and indexing_service
│   ├── extractions/          # Per-page JSON files
│   │   ├── rtgs/
│   │   │   ├── page_001.json
│   │   │   └── ...
│   │   ├── legal_circular_1/
│   │   └── legal_circular_2/
│   ├── crops/                # Visual citation PNG crops
│   │   ├── rtgs/
│   │   └── ...
│   └── catalog.db            # SQLite catalog (indexing audit log)
│
└── legacy/                   # Mount point for my_work/ data (first-run migration)
    └── pipeline_output/      # Symlink or copy of my_work/pipeline_output/
```

## Setup (first run)

```bash
mkdir -p data/docs data/docs_images data/pipeline_output data/legacy
cp -r ../my_work/pipeline_output data/legacy/
cp -r ../my_work/docs_images data/
```
